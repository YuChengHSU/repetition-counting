# repetition-counting
[Paper preprint](https://arxiv.org/submit/3795435/view)
![Alt Text](./Video.gif)

## General usage
        python rep-count.py --data [DATA_dir] 


## Hyper parameters

- j,default=18
    Joint number of the skeleton
- d,default=2
    Dimension of the joint
- f,default=30
    Video frequency
- wins,default=256
    Sliding window frequency
- noverlap"
    Sliding window steps
